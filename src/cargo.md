# Cargo

## Basic commands
```
cargo new
    --bin : binary
    --lib : library
```

```
cargo build ("cargo b")
    --release
```

```
cargo run ("cargo r")
cargo test ("cargo t")
cargo clean
```

```
cargo --version
```
